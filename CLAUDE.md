# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This is a miscellaneous repository for storing random experiments and scripts. There is no unified build system, language, or framework.

## Structure

Each experiment or script lives in its own self-contained directory with:
- A `README.md` explaining what it does
- All necessary code/scripts within that directory
- No dependencies on other directories in this repo

## When Adding New Experiments

- Create a new top-level directory with a descriptive name
- Include a `README.md` in that directory explaining the purpose, usage, and any dependencies
- Keep everything self-contained — do not reference or import from other directories
