# Test Draft PR

This directory was created to verify that the devaipod environment can
successfully open draft pull requests on this repository using the
service-gator MCP tool.

## Purpose

- Validate end-to-end PR creation workflow
- Confirm service-gator GitHub integration works correctly

## What happened

An AI agent running in a devaipod sandbox created this directory, committed
it, and opened a draft pull request — all without using the `gh` CLI.
