# hello-draft-pr

A test experiment to verify that an AI agent running in a devaipod can
successfully open a draft pull request on this repository using the
service-gator toolchain.

## What this tests

- Creating a branch from a local commit
- Pushing the branch to the remote via service-gator
- Opening a draft PR via service-gator's `github_push` tool

## Result

If you're reading this on a PR, it worked.
