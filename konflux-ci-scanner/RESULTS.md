## Detection Methods

**AI review tools** detected via:
1. Config files (`.coderabbit.yaml`, `.pr_agent.toml`, `.gemini/config.yaml`)
2. PR comment search for bot activity (`coderabbitai[bot]`, `qodo-code-review[bot]`, `gemini-code-assist[bot]`)

**CI** detected via:
1. `.tekton/` directory with pipeline YAML files (Konflux)
2. Centralized config in `openshift/release` repo (OpenShift CI)

**Exposed test results** detected via PR comment search for:
- `konflux-ci-qe-bot` -- posts test scenario results with `oras pull` instructions
- `openshift-ci[bot]` -- indicates Prow/ci-operator managing CI with JUnit collection
- `codecov[bot]` / `codecov-commenter` -- coverage reports imply tests are run and results tracked
